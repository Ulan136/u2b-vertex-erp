Шаблоны счетов
Каждый шаблон — это тип счёта с привязанными источниками и банковскими счетами
ℹ️
Шаблон = готовый маршрут: Источник → Тип счёта → Банковский счёт. Менеджер выбирает только конкретный счёт (Каспи/БЦК/Наличка) при каждой операции.

📋 Новый шаблон счёта

1 — Основные данные
2 — Банковские счета (менеджер выбирает один из них)
3 — Источники данных (откуда приходят записи)
4 — Настройки
⚠️ Изменения применяются к новым записям. Существующие данные не затрагиваются.
Каждый филиал — отдельный счёт с независимым журналом.
