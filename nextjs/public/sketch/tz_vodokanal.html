📋 Модуль: Заказы

Данные заказа

  • Номер, дата создания
  • Тип счётчика (ХВС/ГВС/газ)
  • Адрес установки
  • Данные абонента
  • Серийный номер прибора
  • Показания на момент установки

Статусы заказа

  • Новый → Подтверждён
  • В работе → Установлен
  • Принят системой → Закрыт
  • Отменён

Оплата

  • Счёт выставлен
  • Оплачен (частично / полностью)
  • Просроченная задолженность
  • Возврат

Что уходит в Продажи

  • order_id (ключ, без дублей)
  • Сумма и услуги
  • Данные клиента
💰 Модуль: Продажи

Документы

  • Счёт-фактура
  • Договор / акт
  • Кассовый чек

Клиенты

  • Физ. лицо / юр. лицо
  • История покупок
  • Баланс лицевого счёта

Что уходит на Склад

  • sale_id → order_id (ref)
  • Перечень товаров (SKU)
  • Количество к резерву
🏭 Модуль: Склад

Остатки

  • Текущий остаток по SKU
  • Резерв под заказы
  • Свободный остаток
  • Минимальный порог

Движения

  • Приход (от поставщика)
  • Расход (выдача по заказу)
  • Возврат / списание
  • Инвентаризация

Авто-триггер в Закупки

  • Остаток < минимума → alert
  • Создаётся заявка на закупку
  • Без ручного дублирования
🛒 Модуль: Закупки

Заявки

  • Авто-заявка от склада
  • Ручная заявка
  • Согласование руководителем

Поставщики

  • Реестр поставщиков
  • Рейтинг / история
  • Контракты

Приёмка

  • Акт приёмки товара
  • Привязка к склад.приходу
  • Оплата поставщику
⚙️ Модуль: Администрирование

Пользователи & роли

  • Admin, Manager, Warehouse, Buyer
  • Разрешения по модулям
  • Аутентификация NextAuth

Справочники

  • Типы счётчиков
  • Категории товаров (SKU)
  • Единицы измерения
  • Статусные переходы
