=== ПРАВИЛА ЗАГРУЗКИ ===
1. Название — уникальный ключ (после нормализации: трим, схлопнуть
   пробелы, кириллическая К = латинская K при сравнении).
2. Алиасы — варианты написания из реестра; при импорте из Google
   Таблицы значение сначала ищется по алиасам, потом по названию.
3. Стартовый usage_count: Группа 1 = 1000, Группа 2 = 100,
   Группа 3 = 30, Группа 4 = 5, Группа 5 (market) = 0 —
   чтобы подсказки сразу сортировались реалистично.
4. Новые значения, введённые пользователями при сохранении
   сертификата/извещения, добавляются в справочник автоматически
   (самообучение).

Итого: 39 из реестра + 53 рыночных = 92 типа.
